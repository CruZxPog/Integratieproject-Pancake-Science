# The-Golden-Flip
Pancake Science is een educatief IoT Proof of Concept dat STEM-principes uitlegt via pannenkoeken bakken. Een slimme kookplaat meet temperatuur en tijd, logt data op een Raspberry Pi en visualiseert resultaten in een dashboard voor analyse en vergelijking.
